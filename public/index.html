HTML & CSS

Week 1

The first day's assignment was really confusing for me because I hadn't done anything like it before. We had to fix syntax errors and then change HTML classes to match the CSS elements so everything worked correctly.

I didn't know there was a strategy, so I was just guessing. I got worried that if I couldn't do the first assignment maybe I'd made a tiny huge mistake!

HTML & CSS

Week 1

The second assignment was to use the grid system and floats to make Github profiles. We went on Github and found people who worked on projects and used their information in our grids.

This assignment was a lot easier for me to understand than the first one, and I learned about putting things in spans so they're inline from the start.

The third assignment was to do a grid using Octocats, which I really liked. We got to choose which Octocats we used, and since we'd done a similar grid the day before that made it easier to know how to do things.
